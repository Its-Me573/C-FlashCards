# Flash Card Program – First C++ Project

## Purpose
This project was created as my first C++ program to practice core concepts from my first CS class.  
It combines file I/O, vectors, structs, and basic menu-driven programming into a practical application.

## Features
- Store flash cards (question + answer) in a text file (`flashCard.txt`)  
- Read and display flash cards from the file  
- Create new flash cards and save them to the file  
- Menu system to choose between using, creating, or deleting flash cards  
- Useful for studying and practicing concepts  

## Lessons Learned
- Using `struct` and `vector` effectively  
- File input/output (`ifstream` and `ofstream`)  
- Passing vectors by reference to functions  
- Basic program flow and menu-driven interaction  
- Debugging and building a complete, functional program  

## Status
- Fully functional for creating and using flash cards  
- Some minor features (like randomization or deleting cards) can be improved  
- Serves as a completed, useful learning project  

## Notes
This project was a key step in learning C++ fundamentals and building confidence for more advanced projects. It is both a learning milestone and a practical tool for studying.
